These are the different types of coding I have worked with.

HTML5

HTML is an acronym for Hyper Text Markup Language and is one of the most basic ways to code a website.

CSS3

CSS is an acronym for Cascading Style Sheet and adds more cosmetic options to HTML which makes you website look more professional.

BootStrap

BootStrap is a template you can download so you already have several cosmetic features that you just have to code into your website.

JavaScript

JavaScript is a type of coding that lets you code more in-depth things like functions that only trigger after a certain event has occured.
